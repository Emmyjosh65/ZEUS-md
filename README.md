## 📦 Deployment (Pterodactyl)

### Step 1 — Upload the connector

Upload these files:

bash zeus-connector.js package.json 

### Step 2 — Set startup command

bash node index.js 

### Step 3 — Start the server

The connector will automatically fetch, install, and launch the bot.

---

## 🔗 Pair Your Number

Once the bot starts, a pairing code will appear in the console.

Open WhatsApp:

text Settings → Linked Devices → Link with Phone Number 

Enter the pairing code shown in your panel.
