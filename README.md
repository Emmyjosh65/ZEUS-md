## 🚀 What is Zentrix MD?

> **Zentrix MD** is a powerful, feature-rich WhatsApp bot built by **ZENTRIX TECH**, powered by a custom fork of Baileys — [`@zentrix23/baileys`](https://www.npmjs.com/package/@zentrix23/baileys).

It supports **multiple sessions**, **Telegram control**, a **full command suite**, **AI integration**, **group moderation**, **media downloaders**, and much more — all deployable in minutes via Pterodactyl panel.

## 📦 Deployment (Pterodactyl)

### Step 1 — Upload the connector

Upload **only** these two files to your Pterodactyl egg:

```
zentrix-connector.js
package.json
```

### Step 2 — Set startup command

```bash
node zentrix-connector.js
```

### Step 3 — Start the server

The connector will **automatically fetch, install, and launch** the bot. No manual setup needed.

## 🔗 Pair Your Number

Once the bot is running, a **pairing code** will be generated in the console.

Enter it in WhatsApp under:
> **Settings → Linked Devices → Link with phone number**
